## 🎮 Sobre o Jogo

**A Ghost Story: The Death Race** é um jogo arcade 2D desenvolvido em Python com Pygame CE. 
Assuma o controle de um espírito e desvie de inimigos enquanto coleta orbes de energia 
para sobreviver o máximo possível.

### Objetivos
- 🏆 Alcance **2000 pontos** ou sobreviva por **5 minutos**
- 💀 No modo **PESADELO**, a vitória é resistir até o fim
- ⚡ Use o **PowerUp** quando sua energia estiver completa

---

## 🕹️ Controles

| Tecla | Ação |
|-------|------|
| `↑ ↓ ← →` | Movimentar o fantasma |
| `ESPAÇO` (2x) | Ativar PowerUp |
| `ESC` | Voltar / Sair |
| `ENTER` | Confirmar |
